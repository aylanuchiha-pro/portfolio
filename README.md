# Portfolio — Développeur Web & Cybersécurité

Portfolio personnel avec animations 3D, scroll-snap, iframes live, et design éditorial.

## Installation

```bash
npm install
npm run dev
```

Le site sera accessible sur `http://localhost:5173`.

## Build production

```bash
npm run build
```

Les fichiers de production seront dans le dossier `dist/`.

## Personnalisation

- **Infos personnelles** : modifier les textes dans `src/components/Hero.jsx`, `About.jsx`, `Contact.jsx`
- **Projets** : modifier les URLs et descriptions dans `src/components/Projects.jsx`
- **Skills & niveaux** : ajuster dans `src/components/Skills.jsx`
- **Expérience** : mettre à jour dans `src/components/Experience.jsx`
- **Couleurs & fonts** : tout est centralisé dans `src/theme.js`
